# Master-Ontologie

Dies ist die konsolidierte Master-Ontologie-Repository, die alle relevanten Definitionen, Entitäten und Klassifikationen enthält.

## Struktur

- `master_ontology.yaml` - Die zentrale Ontologie-Datei mit allen Definitionen
- - `Elevation_Entities.yaml` - Spezialisierte Entitäten für das Elevation-Projekt
  - - `Elevation_Uebergabe.md` - Dokumentation zum Übergabeprozess
   
    - ## Beschreibung
   
    - Diese Repository konsolidiert alle vorherigen Ontologie-Entwicklungen und ist die Single Source of Truth für die Master-Ontologie.
